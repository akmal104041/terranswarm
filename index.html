TerranSwarm

This page contais information on TerranSwarm bot. It is a Terran StarCraft/BWAPI bot that uses a task allocation algorithm to decide its actions. The parameters of task allocation were adjusted with a Genetic Algorithm in matches against StarCraft's native AI.
